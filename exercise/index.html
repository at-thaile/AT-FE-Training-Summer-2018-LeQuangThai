Question 1: Sự khác nhau giữa null và undefined

Trả lời : Biến được khai báo nhưng không được gán giá trị thì có giá trị mặc định là undefined
      typeof null          // "object" (not "null" for legacy reasons)
      typeof undefined     // "undefined"
      null === undefined   // false
      null  == undefined   // true
      null === null        // true
      null == null         // true
      !null                // true
      isNaN(1 + null)      // false
      isNaN(1 + undefined) // true

What are the differences between `==` and `===`? Write an example for each case (if any)?

    Equal(==) :Returns true if the operands are equal.
    Equal(===):Returns true if the operands are equal and of the same type
    Example: 3== "3" //true
             var s = "3";
             3 === s; //false

What is different between declaration: `var`, `const` and `let`?

    var: Declar a variable has a cross through range range contains it.
    let: Declare a variable that is only accessible in the block surrounding it.
    const: Declaring a constant - is an unchanging value during the run.
